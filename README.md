# AI Study Assistant

An AI-powered web application that helps students upload study notes, generate AI summaries, and prepare for exams.

## Features

-  User Registration & Login
-  Secure Password Hashing
-  PDF Upload
-  PDF Text Extraction
-  AI-Powered Summary using Google Gemini
-  User Dashboard
-  Logout

## Tech Stack

- Python
- Flask
- SQLite
- SQLAlchemy
- HTML
- CSS
- PyPDF2
- Google Gemini AI
